# rc-typing-effect
Make typing effect for your text!

## Installation
```
yarn add rc-typing-effect or npm install rc-typing-effect --save
```

## Props
```
/** string */
message: '',
/** Delay in millisecond */
deplay: 0,
/** Character per millisecond */
cpms: 125,
```
