# rc-typing-effect
Make typing effect for your text!

## Installation
```
yarn add rc-typing-effect or npm install rc-typing-effect --save
```

## Props
```
/** Set Root element to span, h1, h2, ... */
as: 'span',
/** Text */
text: ''
/** Last blinking duration in second */
delay: 3
/** Character per second */
cps: 5
style: {}
className: ""
```
